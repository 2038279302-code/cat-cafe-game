## 📖 游戏简介

你是一家温馨猫咪咖啡馆的店长！顾客们陆续上门，你需要先选好要制作的菜品，再点击顾客接单、开始烹饪。动作要快——顾客耐心有限，别让他们等太久！

随着经营等级提升，新菜品陆续解锁；用赚到的金币升级制作台、咖啡机和等候区；多摸摸猫咪还能解锁专属技能，让猫咪帮你打工！

---

## 🕹️ 操作方式

| 操作 | 说明 |
|------|------|
| **① 选择菜品** | 点击左侧菜单选好要做的品类 |
| **② 接待顾客** | 查看顾客头顶气泡，确认 TA 想要什么 |
| **③ 点击制作** | 菜品匹配后点击顾客，分配制作台开始烹饪 |
| **④ 多台并行** | 解锁多个制作台后，可同时接多个订单！ |
| **🛒 升级商店** | 用金币升级制作台、咖啡机、等候区 |
| **摸摸猫咪** | 积累亲密度，解锁猫咪专属技能 😺 |

> **手机玩家**：直接用手指点击操作，界面已适配移动端 📱

## 🛠️ 项目结构

```
cat-cafe-game/
├── index.html        # 页面主文件
├── css/
│   └── style.css     # 样式（含移动端响应式 + 商店面板 + 技能UI）
└── js/
    ├── data.js       # 游戏常量（菜品、成就、升级配置、技能配置）
    ├── state.js      # 全局状态（多制作台、商店升级、猫咪技能）
    ├── draw.js       # Canvas 绘图（场景、顾客、猫咪、多进度条）
    ├── save.js       # 本地存档（历史最佳 + 商店/技能跨局持久）
    ├── ui.js         # UI 更新（商店面板、亲密度、技能解锁）
    └── game.js       # 核心逻辑（多台并行、猫咪帮厨、调度策略）
```
