## 📖 游戏简介

你是一家温馨猫咪咖啡馆的店长！顾客们陆续上门，你需要先选好要制作的菜品，再点击顾客接单、开始烹饪。动作要快——顾客耐心有限，别让他们等太久！

随着经营等级提升，新菜品陆续解锁，还有 VIP 顾客和急性子客人带来更多挑战与奖励。

---

## 🕹️ 操作方式

| 操作 | 说明 |
|------|------|
| **① 选择菜品** | 点击左侧菜单选好要做的品类 |
| **② 接待顾客** | 查看顾客头顶气泡，确认 TA 想要什么 |
| **③ 点击制作** | 菜品匹配后点击顾客，开始烹饪 |
| **④ 别让等太久** | 进度条变红就要赶快完成订单！ |
| **摸摸猫咪** | 点击左侧「摸摸猫咪」按钮，提升猫咪心情 😺 |

> **手机玩家**：直接用手指点击操作，界面已适配移动端 📱

## ✨ 特殊顾客

- **👑 VIP 顾客**（20% 概率）：耐心 ×1.5，完成后额外给小费
- **⚡ 急性子顾客**（20% 概率）：只等 ~10 秒，但完成订单**奖励翻倍**

---

## 🔥 连击系统

连续完成订单可触发连击加成：

| 连击数 | 金币倍率 |
|--------|----------|
| 3 连击 | ×1.2 |
| 5 连击 | ×1.5 |
| 10 连击 | ×2.0 |

连击会在 6 秒内没有新订单完成时重置。

## 💾 存档功能

游戏成绩自动保存在本地（`localStorage`），关闭页面后再次进入可在首页看到**历史最佳记录**（最高金币、最高连击、最高等级、最佳评级）。

## 🛠️ 项目结构

```
cat-cafe-game/
├── index.html        # 页面主文件
├── css/
│   └── style.css     # 样式（含移动端响应式适配）

```
